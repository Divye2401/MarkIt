# Markit Extension - Privacy Policy (Chrome Web Store Version)

## What We Collect

- **Authentication**: Google account info (email, name, profile picture) for login
- **Bookmarks**: URLs, titles, and content you save using the extension
- **AI Data**: Webpage content temporarily processed to generate summaries and tags
- **Usage**: Basic analytics to improve the extension

## How We Use Your Data

- Store and sync your bookmarks across devices
- Generate AI-powered summaries and tags using OpenAI
- Provide intelligent search and organization features
- Improve extension performance and features

## Data Security

- All data encrypted and stored securely in Supabase
- User data is completely isolated (you only see your bookmarks)
- No data sold to third parties
- Optional analytics can be disabled in settings

## Third-Party Services

- **Google OAuth**: For secure authentication
- **OpenAI**: For AI content analysis (data not retained by OpenAI)
- **Supabase**: For secure data storage and user authentication

## Your Rights

- Access, edit, or delete your data anytime
- Export all your bookmarks
- Delete your account and all associated data
- Disable analytics and AI features in settings

## Permissions

- **Active Tab**: To read page titles/URLs when saving bookmarks
- **Storage**: To store login tokens and settings locally
- **Host Permissions**: To sync with Markit servers
